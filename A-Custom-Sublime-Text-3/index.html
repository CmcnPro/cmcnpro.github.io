一个即开即用的定制sublime3
2017-09-14

2017年9月13日,Sublime Text 3 is out!之前一直是使用他人定制捆绑包,但是一直觉得他们的定制的东西有点多,于是,趁着新版本的发布,自己定制一个适合自己的简单版本.

修改日志

  • 基于2017年9月13日发布的Build 3143(x64)版本定制(Only Windows);
  • 安装包管理器 Package Control;
  • 安装语言包ChineseLocalizations,默认显示简体中文Simplified Chinese,可通过帮助-language切换其他语言;
  • 安装ConvertToUTF8,提高语言支持;
  • 安装OmniMarkupPreviewer,右键即可在浏览器预览Markdown;
  • 安装SideBarEnhancements,菜单增强;
  • 安装ayu主题,并默认启用;
  • 修改菜单,改回以前的设置-默认和设置用户选项,并且在当前页面打开(修改语言后会失效);
  • 增加用户文件参数;
 {
    "color_scheme": "Packages/ayu/ayu-light.tmTheme",
    "font_options":
    [
        "gdi"
    ],
    "font_size": 11,
    "ignored_packages":
    [
        "Vintage"
    ],
    "index_files": true,
    "open_files_in_new_window": false,
    "theme": "ayu-light.sublime-theme",
    "update_check": false,
    "word_wrap": "auto"
}
